<h2>一、项目简介</h2>
基于ssm的NBA网商平台管理系统：前端jsp、jquery、bootstrap，后端 spring、mybatis，集成商品管理、订单管理、资讯管理、商品前台展示、商品详情、分类展示、购物车等功能于一体的系统。
